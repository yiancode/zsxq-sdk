# UI 交互参考

此目录存放知识星球App的UI截图，用于API设计和业务流程分析参考。

## 目录结构

- `planet/` - 星球相关（列表、详情、设置等）
- `topic/` - 话题相关（发布、编辑、评论等）
- `training-camp/` - 训练营相关（打卡、排行榜等）
- `member/` - 成员管理相关（星主功能）

## 命名建议

使用描述性文件名，便于查找：
- `planet-list.png` - 星球列表页
- `planet-detail.png` - 星球详情页
- `topic-create-step1.png` - 创建话题步骤1
- `checkin-flow.png` - 打卡流程
